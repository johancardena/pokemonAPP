# PokeApp 

## Descripción

Aplicación desarrollada con **Ionic Angular** que permite buscar Pokémon por nombre utilizando la API pública **PokeAPI**.

El usuario puede ingresar el nombre de un Pokémon y visualizar su información básica junto con su imagen.

##  Funcionalidades

* Búsqueda de Pokémon por nombre
* Visualización de imagen
* Información del Pokémon:
  * ID
  * Altura
  * Peso
  * Tipos
  * Habilidades

## Tecnologías utilizadas
* Ionic
* Angular
* TypeScript
* PokeAPI
